# Company Lens

企業を、鑑賞するための日本株評価アプリのプロトタイプです。

## 使い方

1. ZIPを展開
2. `index.html` をダブルクリック
3. ブラウザで開く

## 今できること

- 左サイドバー付きのアプリUI
- 銘柄コード・企業名検索
- 企業一覧
- 企業詳細カード
- スコアリング表示
- 指標バー
- AI Insight風コメント

## 次にいじる場所

### 企業データを増やす

`app.js` の `companies` 配列に追加します。

### 見た目を変える

`style.css` を編集します。

### 実データ化する

今はサンプルデータです。次の段階で、財務データAPIやCSV読み込みに差し替えます。

## 方向性

Yahoo!ファイナンスや掲示板ではなく、Bloomberg / Linear / Apple のような静かで洗練された企業評価アプリを目指します。
